# Technologies Utilisées

## Stack Core
- **Node.js / Python** : Langages d'exécution pour les compétences de l'agent.
- **OpenClaw** : Framework d'Agents IA autonome.
- **Model Context Protocol (MCP)** : Standard pour relier le LLM aux APIs de courtage (via `alpaca-mcp-server`).
- **Alpaca API** : API de courtage pour US Stocks & ETFs.

## Infrastructure
- **VPS Linux** (Ubuntu recommandé pour stabilité continue).
- **Git** : Versioning métier.
