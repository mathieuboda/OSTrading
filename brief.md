# OpenClaw Alpaca Agent - Foundation Brief

## Core Requirement
Développer un agent de trading IA basé sur OpenClaw (et le Model Context Protocol) pour opérer sur la bourse traditionnelle (US Stocks) et le Forex, via le courtier Alpaca. 

## Goals
L'objectif est d'exploiter les avancées des LLM pour l'analyse en temps réel (actualités, web, réseaux sociaux) afin de prendre des décisions d'investissement automatisées, le tout hébergé de manière sécurisée sur un VPS. On exclut fermement le trading crypto.
