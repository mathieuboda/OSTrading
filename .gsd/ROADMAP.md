# ROADMAP.md

- [ ] Hébergement VPS sécurisé (sandbox).
- [ ] Installation du framework de base OpenClaw.
- [ ] Connexion fonctionnelle à l'API Alpaca Paper Trading.

**Status**: ⬜ Not Started
**Objective**: Configurer le VPS, sécuriser l'environnement et installer OpenClaw (via MCP/Alpaca) sans aucune exposition financière.

### Phase 2: Alpaca Skills Integration
**Status**: ⬜ Not Started
**Objective**: Interfacer OpenClaw avec le serveur MCP Alpaca et valider les appels API (lecture du compte, ordres virtuels).

### Phase 3: Sentiment Analysis Pipeline
**Status**: ⬜ Not Started
**Objective**: Permettre à l'IA d'interroger le web (news, réseaux sociaux) avant d'exécuter un ordre.

### Phase 4: Full Simulation (Paper Trading)
**Status**: ⬜ Not Started
**Objective**: Lancer l'agent en mode autonome sur le marché de démonstration et logger ses performances / rationale de décision.
